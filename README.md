## 📖 About

**Debug Lab** is a browser-based debugging simulator designed to sharpen your frontend development skills through hands-on practice. You're given broken code — CSS layouts that don't work, JavaScript logic errors, console errors to decode, and more — and your job is to find and fix the bug.

No frameworks. No build tools. No installations. Just open `index.html` and start debugging.

> **Built for developers who learn by doing.**

## ✨ Features

| Feature | Description |
|---|---|
| 🎮 **10 Game Modes** | CSS, JavaScript, Console Errors, DOM, Async, HTML, React, TypeScript, API, Regex |
| 🧩 **73 Challenges** | From easy warmups to hard brain-teasers |
| 👁️ **Live Preview** | CSS and HTML changes render instantly in an iframe |
| 🖥️ **Console Panel** | Shows error clues, run results, and hint messages |
| ⏱️ **Timer & Scoring** | Time bonuses for fast solves, penalties for hints |
| 💡 **Progressive Hints** | 3 hints per challenge, each costs more points |
| 🔥 **Streak System** | 3+ consecutive solves earn bonus points |
| 📊 **Grade System** | S, A+, A, B, C, D, F grades based on accuracy |
| 📋 **Challenge Breakdown** | Detailed results for each challenge after mode completion |
| 🔊 **Sound Effects** | Web Audio API beeps for success, error, hint, skip |
| 💾 **Persistent Stats** | High scores, mode bests, and total bugs fixed saved in localStorage |
| 📱 **Responsive Design** | Works on desktop, tablet, and mobile |
| ⌨️ **Keyboard Shortcuts** | `Ctrl+Enter` to run, `Escape` to exit, `Tab` for indentation |
| 📤 **Share Results** | Copy your results to clipboard |
| 🔢 **Line Numbers** | Editor has a synced line number gutter |
| 🎨 **Professional UI** | Clean dark theme with Inter + JetBrains Mono fonts |
